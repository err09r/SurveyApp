# Survey App

![](https://img.shields.io/badge/Status-Work%20in%20Progress-yellow)

Simple app for conducting A/B testing built with Kotlin and Jetpack Compose.

# 

 ### Tech-stack
- **UI:** Jetpack Compose
- **Async:** Coroutines
- **DI:** Hilt
- **Backend:** Firebase APIs (Authentication, Realtime Database, Storage)
